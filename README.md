# Soran-skills

自用 skills 集合仓库。

这个仓库的目标不是把 skill 按目录堆起来，而是把它们整理成一个更容易浏览、分享和回看的集合首页。

## Skills

### `parable-concept-explainer`

把研究生水平的概念写成寓言式故事，先埋伏笔，后揭示概念名，最后再用短解释把概念对齐清楚。

- 适合：想要“不要直说，绕着讲，最后再揭晓”的概念讲解
- 特点：故事优先，术语后置，信源先校准事实
- 文件：
  - [`README.md`](parable-concept-explainer/README.md)
  - [`SKILL.md`](parable-concept-explainer/SKILL.md)

### `Soran-free-image-search`

搜索可免费商用图片的 skill。

- 适合：找配图、下载可商用图片
- 文件夹：
  - [`Soran-free-image-search/`](Soran-free-image-search/)

## 结构

```text
Soran-skills/
├── README.md

```

## 使用方式

- 直接在 GitHub 打开仓库首页查看
- 点进具体 skill 文件夹看 `README.md` 和 `SKILL.md`
- 如果你想把某个 skill 单独发出去，可以再单独拆成一个仓库

## 说明

这个首页会随着新 skill 增加而更新。它更像目录和展示页，不是完整教程。
